What is Build Watch?

Think of it like Lovable’s preview pane. Grok writes code in your terminal; this page shows the running app so you don’t stare at logs.

Link Grok
Connects to your active Grok Build session so we can see edits and terminal output.
Your app preview (center)
The thing you care about — your site or UI. Refreshes when Grok changes HTML or when a dev server is running.
What Grok did
A readable timeline: files edited, commands run. Click any file path.
Open a file
Shortcuts to HTML pages and docs in your project.
Terminal at the bottom
Optional mirror of Grok’s terminal. You don’t type here unless you expand it and switch modes — coding stays in Grok.

Use in any tab

  • Browser: Keep http://127.0.0.1:8790 open in any tab (pin it). Use Copy link in the header.
  • Grok terminal: You can have several Grok tabs. Use the Grok tab dropdown to choose which one Build Watch follows, then Link Grok.
  • CLI: From your project: build-watch on then build-watch connect or build-watch connect <session-id>
  • Deep link: ?session=YOUR_SESSION_ID on the URL auto-links when the page loads.

Always start the server from your project: cd your-project && build-watch on
